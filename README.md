# cash.github.io

Clark's personal space — built with plain HTML, CSS, JS.

## Setup on GitHub Pages

1. Go to github.com → New repository
2. Name it exactly: `cash.github.io`
3. Set it to **Public**
4. Upload all 3 files: `index.html`, `style.css`, `script.js`
5. Go to Settings → Pages → Source: `main` branch → `/root`
6. Your site will be live at: **https://cash.github.io**

## File structure

```
cash.github.io/
├── index.html   ← structure
├── style.css    ← deep space theme
├── script.js    ← stars + interactions
└── README.md
```

## Editing

- Add your real social links in `index.html` (search for `href="#"`)
- Add projects in the `.cards` section
- Drop files in the Vault section as `<div class="file-item">` blocks
